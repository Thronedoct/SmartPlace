# Model Experiments

Experiment scripts and lightweight configuration files live here. Large datasets, raw checkpoints, and generated artifacts should stay under ignored data/model folders or `report/`.

Planned experiments:

- `opa_baseline/`: run OPA/libcom baseline scoring.
- `opa_rgb_mask/`: compare RGB scoring against RGB+mask scoring.
- `opa_finetune/`: optional small-subset fine-tuning.
- `explainability/`: Grad-CAM or occlusion/sensitivity analysis.
