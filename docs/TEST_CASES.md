## 最终覆盖要求

最终项目至少准备 18 组案例，覆盖：

- 桌面或台面场景。
- 地面场景。
- 墙面、架子或柜体场景。
- 户外场景。
- 明显合理的位置。
- 明显错误的位置。
- 边界越界或尺度不合理的位置。

## 每个重点案例建议保存的证据

- 原始背景图。
- 前景物体图。
- Top 3 候选展示截图。
- 模型评分表。
- 人工判断。
- 如果结果不好，需要写清失败原因。
